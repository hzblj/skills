# Commands

Custom commands for this plugin.

- [**/audit-imports**](audit-imports.md) — Audit the monorepo against the project structure boundaries and dependency direction
- [**/clean-review**](clean-review.md) — Review the current git diff against the repo's clean-code + type-safety standards
- [**/new-component**](new-component.md) — Scaffold a component following component-architecture + clean-code standards
- [**/new-feature**](new-feature.md) — Scaffold a new feature-* package following the project (monorepo) skill
- [**/orchestrate**](orchestrate.md) — Plan a task, then delegate across the tiered agents (deep-reasoner, domain builders, fast-worker) and synthesize the results
- [**/polish**](polish.md) — Apply an interface-polish pass to a component (web or React Native)
- [**/refactor**](refactor.md) — Refactor a file or area to the repo's clean-code + type-safety standards, preserving behavior
